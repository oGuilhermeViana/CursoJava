# Como declarar varialvel em Java

- <tipo>  <nome da varialvel>;
- <tipo>  <nome da varialvel> = <valor>;

# Convenção de nomenclatura

- primeira letra
    * A-Z, a-z, _ ou &

-segunda em diante
    * A-Z, a-z, _, & ou 0-9

- camelCase
    * nomeCompleto

# Tipos primitivos

- String
- Char
- Int
- Double
- Float
- boolean
